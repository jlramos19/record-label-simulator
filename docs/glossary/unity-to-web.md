# Unity to Web Glossary

Concise mappings for retconning Unity-era terminology into web equivalents.

## Mappings
- Unity Entity Debugger -> Debug/Inspect Mode + browser console logging
- Unity Inspector -> in-app debug panels / read-only inspectors
- ECS tooling -> deterministic data model + event log inspection
- GameObject / MonoBehaviour -> TypeScript modules with state and update helpers
- ScriptableObject -> JSON or TS data modules loaded at runtime
- Prefab -> UI template or data-driven component config
- Unity console -> browser console
