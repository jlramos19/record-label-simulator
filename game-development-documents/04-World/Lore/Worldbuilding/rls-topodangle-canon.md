# **RLS Topodangle — Canon Specification**

This document describes the exact canonical orientation, coordinate system, and plotting rules for the RLS Topodangle.

AXIS ORIENTATION

• X axis \= horizontal (left negative, right positive)

• Y axis \= depth (forward negative, backward positive)

• Z axis \= vertical (down negative, up positive)

VERTEX ASSIGNMENTS (Neutral Profile)

• Freedom  \= ( 0.0,  0.0, \+0.5)

• Power    \= ( 0.0,  0.0, \-0.5)

• Loyalty  \= ( 0.0, \+0.5,  0.0)

• Morality \= (+0.5,  0.0,  0.0)

• Ambition \= (-0.5,  0.0,  0.0)

NOTES

• Loyalty must always sit “behind” (positive Y).

• Morality must always be right (positive X).

• Ambition must always be left (negative X).

• Freedom must always be above (positive Z).

• Power must always be below (negative Z).

• All five vertices are equidistant from the origin, forming a symmetric tri‑pyramid.

• This orientation is canonical and must never be altered.
