Tutorial

Start here: sign Creator IDs in the CCC, build a track (Sheet -> Demo -> Master), release it, and fund a promo push. Use time controls to move weeks and watch charts climb.
Game Loops
Sign Creator IDs (Community -> CCC)
Refresh the pool and sign at least one Songwriter, Recorder, and Producer.
Form an Act (Harmony Hub - Acts)
Assign Creator IDs to Act slots to prep releases and eras.
Create content (Create view)
Assign Creator IDs, pick Theme/Mood, then run Sheet -> Demo -> Master stages.
Schedule release (Calendar)
Choose distribution; Digital is fast, Physical or Both pays more.
Promote (Promotions -> eyeriSocial)
Pick a Project or scheduled/released Track ID, set budget, and fund promo pushes. Promotions allowed once Scheduled. Unreleased: schedule first.
Plan tours (Touring)
Select an Act + Era, pick an anchor release, set a window, then book venues from the Venue Finder.
Weekly cadence (168 in-game hours)
Release processing Friday 00:00, trends Friday 12:00, charts Saturday 00:00. Use Pause/Play/Fast and Skip Time to move weeks.
Game Concepts
Creator signing decisions
CCC acceptance depends on label focus Themes/Moods, creator skill, and market heat. Rejections log a reason and lock until the 12AM refresh.
Trends guide demand
Check Top Trends in Community/Create and follow recommendations.
Studios and stamina are bottlenecks
Active Studios cap parallel work; bigger teams are faster but drain stamina.
Rent studio capacity when you outgrow slots
Exceed owned studios to auto-lease market slots; lease fees add to weekly upkeep.
Eras focus your cadence
Use Era Desk to set a Focus Era for creation and promo priorities.
Cashflow matters
Watch Wallet for weekly revenue/upkeep and avoid negative cash.
CEO Requests (achievements)
Track your 12 Requests in the Dashboard and race rivals to break records before year 3000.
Interface tips
Use ID slots or drag-and-drop, Auto Assign/Recommend All to fill quickly, customize panels, and save often.
