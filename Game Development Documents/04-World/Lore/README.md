# Lore Index

This folder centralizes lore materials for quests, narrative, and world flavor.

## Subfolders
- `Characters` - Character bios, quotes, and family notes.
- `Dialogue` - Dialogue scripts and line collections.
- `Lyrics` - Track lyrics and lyric notes.
- `Names` - Name lists and naming lore.
- `Story` - Story beats, scenes, and narrative drafts.
- `Worldbuilding` - Mythology, history, and canon references.
