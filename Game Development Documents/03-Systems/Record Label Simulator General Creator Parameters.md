# **Record Label Simulator — General Creator Parameters**

## **Purpose**

Defines the universal parameters for Creators in Record Label Simulator. These values guide design, simulation behavior, and worldbuilding consistency.

## **General Creator Parameters**

| Parameter | Options/Description |
| :---- | :---- |
| Age | 10–100 years old |
| Gender/Demeanor | Flamboyant · Androgynous · Modest |
| Continent of Origin | North American · South American · European · African · Asian · Oceanian |
| Occupation | Songwriter · Recorder · Producer |
| Alignment | Safe · Neutral · Risky (affects Audience trust, Charts velocity, and Critics’ perception) |
| Preferred Theme | Freedom · Loyalty · Ambition · Morality · Power |
| Preferred Mood | Cheering · Saddening · Thrilling · Angering · Calming · Energizing · Uplifting · Boring · Daring |
| Hairstyle | Varies by age, content genre, and country of origin |
| Facial Features | Reflect diverse continental backgrounds |
| Wealth Level | Low to high (Creator success \+ inherited wealth) |
| Environment | Creators pictured in locations reflecting preferences \+ wealth level |
