# Economy - Country Currencies and Acronyms

This document defines country and currency abbreviations used in Gaia, plus the default currency name and usage notes.

## Country Acronyms
- Annglora: AN
- Bytenza: BY
- Crowlya: CR

## Currency Names and Uses
- Annglora: Dollar (primary reference currency for global transactions)
- Bytenza: Won (official currency for local transactions in Bytenza)
- Crowlya: Peso (official currency for local transactions in Crowlya)

## Currency Acronyms and Symbols
- Annglora Dollar: AND$ (AND)
- Bytenza Won: BYW (BYW)
- Crowlya Peso: CRP (CRP)
